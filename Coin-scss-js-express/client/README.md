# Дипломная работа на js Тодор О.В.

## Установка
* Установите [Node.js](https://nodejs.org/en/download/)
`npm i`
* Установите [yarn](https://yarnpkg.com/lang/en/docs/install/) 
`npm install --global yarn`
* Установите зависимости проекта 

## Для разработки:
`yarn start`

## Сборка:
`yarn build`
